Tokenisation is of 3 types:
Word Based, Character Based, SubWord Based. 

Word Based Tokenisation converts each word into a seperate token:
Drawbacks: Very large vocubalary, large quantity of out of quantity tokens, loss of meaning across similar words.

Character Based Tokenisation converts each character into a seperate token:
Drawbacks: Very Long sequences, Less meaningful individual tokens.

SubWord Based Toeknisation: Takes best of both, keep base words as tokens, prefixes and sufixes as tokens.
For e.g. Dogs -> Dog + s
